Architecture

6-Layer Stack

The unified system connects four repositories through the nucleus as the universal algebraic structure:

  1. Semantic Layer: PrimaryAlgebra (Frame), Nucleus, Reentry, BoundaryHeyting
  2. LoF Kernel: Expr → BDD → MuxNet → GateSpec → AIG
  3. SKY Combinators: K/S/Y reductions, bracket abstraction, multiway
  4. Topos Layer: Sieves, Grothendieck topology, J.close nucleus
  5. Compilation: LambdaIR → MiniC → C
  6. Knowledge Layer: TensorLogic (boolean/f2/fuzzy/heyting modes)
